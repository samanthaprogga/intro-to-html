Size Units Example

1pc

One pica (pc) is approximately 1/6th of an inch. It is commonly used in print design.

10pt

One point (pt) is equal to 1/72nd of an inch. It is often used in typography and print.

12px

Pixels (px) are a relative unit commonly used for screen displays. 1px is one pixel on the screen.

2em

The "em" is a scalable unit, and in this context, 2em means twice the size of the parent element's font size.

3rem

The "rem" unit is similar to "em" but is relative to the root (html) element. 3rem is three times the root font size.

50%

Percentage (%) is a relative unit that refers to a percentage of the parent element. 50% takes half the size of its parent.

20vh

Viewport height (vh) is a percentage of the viewport's height. 20vh takes 20% of the viewport's height.
